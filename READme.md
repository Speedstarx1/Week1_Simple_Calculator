# Simple C# Calculator

A simple console-based calculator written in C#.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Input validation using `TryParse`

## Technologies
- C#
- .NET Console Application

## Purpose
This project was built as part of learning C# and .NET fundamentals.

## How to Run
1. Clone the repository
2. Open in Visual Studio or VS Code
3. Run the project
